### ✅ 第 8 步：滤镜系统

**目标**：滤镜面板，选择滤镜应用到合成图，强度可调

**前置依赖**：第 7 步

**操作**：
- [x] 滤镜面板（XP 浮动工具栏样式，分类显示）
- [x] XP老数码：CCD相机 / 复古胶片 / 低清像素
- [x] Y2K霓虹：冰蓝冷调 / 粉紫柔光
- [x] 滤镜强度滑块（0% ~ 100%）
- [x] 选中滤镜 → 像素级处理（getImageData/putImageData）
- [x] 防抖处理：拖动滑块 100ms 后才重新渲染
- [x] 「无滤镜」选项（恢复原始画面）
- [x] 强度混合：globalAlpha 叠加滤镜图与原图

**滤镜算法**：
- CCD相机：降低饱和度 30% + 随机噪点 ±9 + 色温偏暖(R+8, B-6)
- 复古胶片：对比度×1.2 + 暗角(vignette) + 暖黄色调
- 低清像素：缩放至 1/4 → 最近邻放大（imageSmoothingEnabled=false）
- 冰蓝冷调：R×0.85, B×1.15, G×0.95
- 粉紫柔光：R×1.1, B×1.08, 对比度降低 15%

**验证标准**：✅ 切换滤镜实时生效，强度调节流畅
