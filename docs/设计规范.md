# 「XP复古人生四格照相馆」设计规范

> 版本：v1.0 | 日期：2026-07-13

---

## 一、设计原则

> **XP 系统基底 + Y2K 千禧复古融合 —— 所有视觉元素先像 XP，再点缀 Y2K。**

- 功能性界面沿用 XP 原生配色和控件样式
- 装饰性元素使用 Y2K 霓虹/镭射/像素风格
- 两者不冲突：XP 负责「能用」，Y2K 负责「好看」

## 四、字体方案

### 系统功能文字（XP 原生风格）

```css
font-family: 'Microsoft YaHei', '宋体', 'SimSun', 'NSimSun', serif;
```

| 元素 | 字体 | 大小 | 粗细 |
|------|------|------|------|
| 窗口标题 | 微软雅黑 | 13px | bold |
| 正文/标签 | 宋体 | 12px | normal |
| 按钮文字 | 宋体 | 12px | normal |
| 状态栏 | 宋体 | 11px | normal |
| 任务栏时间 | 微软雅黑 | 11px | normal |

### 装饰像素文字（Y2K 风格）

```css
font-family: 'Courier New', 'FangSong', monospace;
/* 备用方案：在等宽字体上使用 letter-spacing 模拟像素感 */
image-rendering: pixelated;
```

| 元素 | 字体 | 大小 |
|------|------|------|
| 按钮小标题 | 等宽像素字体 | 14px |
| 倒计时数字 | 等宽像素字体 | 48px |
| Y2K 装饰文字 | 等宽像素 + 霓虹发光 | 16-24px |

## 六、动效参数

| 动效 | 时长 | 缓动函数 | 说明 |
|------|------|----------|------|
| 窗口打开 | 200ms | ease-out | 从中心缩放弹出 |
| 窗口最小化 | 200ms | ease-in | 缩小消失到任务栏 |
| 按钮悬浮 | 100ms | linear | 背景浅蓝高亮 |
| 按钮按下 | 50ms | linear | 内凹阴影 |
| 弹窗出现 | 150ms | ease-out | 轻微缩放 + 淡入 |
| 滤镜切换 | 300ms | ease | 溶解过渡 |
| 漂浮闪片 | 8-12s | linear | 无限循环从上到下飘落 |
| 倒计时数字 | 1s/次 | steps(1) | 像素帧切换 |
| 像素加载沙漏 | 无限 | steps(4) | 4帧循环动画 |

## 七、装饰元素规格

### 漂浮物（16bit 像素风格）

- 元素类型：爱心 💗、蝴蝶 🦋、五角星 ⭐、闪片 ✨、蝴蝶结 🎀
- 大小：12×12px ~ 20×20px
- 透明度：20%~60%（半透明不遮挡操作）
- 运动：随机起始 X，从上到下匀速飘落，8-12s 完成一次
- 数量：同时可见 10-15 个

### 分割线

- 像素虚线：`2px dashed #ACA899`
- 镭射渐变线：`linear-gradient(90deg, #A855F7, #EC4899, #2DD4BF)`
- 高度：2px

### 图标风格

- 所有功能图标使用 CSS 绘制（避免加载外部图片）
- 像素风格 16×16 ~ 32×32
- 颜色限制在 16 色调色板内（模拟 16bit 色深）
