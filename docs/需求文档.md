# 「XP复古人生四格照相馆」需求文档

> 版本：v1.0 | 日期：2026-07-13 | 状态：Phase 1 开发中

---

## 一、项目背景与目标

### 背景
用户希望在 Windows 电脑上拥有一个「线上拍贴机」工具——像韩国人生四格拍照亭一样，可以拍照、排版、加滤镜、导出四宫格照片。整体界面完全复刻 Windows XP 操作系统风格，叠加千禧年 Y2K 甜酷复古美学。

### 目标
一个**纯前端单 HTML 文件**，双击用浏览器打开即可使用，无需安装任何软件，不上传任何数据到云端。

---

## 二、用户画像

- 不懂代码的普通用户
- 使用 Windows 电脑（台式机/笔记本）
- 用 Chrome 或 Edge 浏览器
- 喜欢复古 Y2K 风格、喜欢拍照分享

---

## 三、功能需求（6大模块）

### 模块一：素材导入（双模式）

| 功能 | 详细说明 |
|------|----------|
| 电脑摄像头拍照 | 实时预览、倒计时(3s/5s/10s)、单张/连拍4张、快门音效 |
| 本地上传图片 | 点击选择文件、拖拽导入、支持JPG/PNG/WebP |
| 缩略图陈列 | 导入后自动生成缩略图，最多同时展示4张素材 |

### 模块二：模板库

| 分类 | 模板 |
|------|------|
| 人生四格基础款 | 标准2×2横/竖四格、长条横向四格 |
| 人生四格创意款 | 错位不规则四格、爱心圆角四格、波浪花边四格、像素网格描边四格 |
| 边框样式 | 复古黑胶片边、奶油白宽边、银质镭射金属边、碎花花边、XP像素格边框 |
| 自定义参数 | 图片间距、边角圆角大小、边框粗细 |

### 模块三：滤镜系统

| 分类 | 滤镜 |
|------|------|
| XP老数码 | CCD相机、老式电脑摄像头、复古柯达/富士胶片、低清像素模糊 |
| Y2K千禧霓虹 | 冰蓝冷调、粉紫镭射柔光、低饱和奶油风、金属反光 |
| 日系清新 | 冷白清透、暖调奶油、低饱和柔光 |
| 附加特效层 | 胶片噪点、划痕、雪花颗粒、复古暗角 |

### 模块四：修图工具

| 功能 | 说明 |
|------|------|
| 基础调色 | 亮度、对比度、色温、饱和度、锐化、模糊 |
| 裁切变换 | 自由裁剪、90°旋转、水平/垂直翻转 |
| 像素贴纸库 | 爱心、蝴蝶、星星、镭射闪片、千禧艺术字、XP图标 |
| 文字涂鸦 | 像素手写字体、霓虹描边文字、荧光笔/粉笔笔刷 |
| 复古美颜 | 磨皮、肤色提亮、淡腮红 |

### 模块五：本地相册

| 功能 | 说明 |
|------|------|
| 分类文件夹 | 我的四格成品、拍立得胶片、原始拍摄素材 |
| 相册操作 | 网格缩略图、全屏预览、删除、重命名、标签 |
| 批量导出 | 导出至桌面/图片文件夹 |
| 二次编辑 | 重新导入编辑界面，更换模板滤镜 |

### 模块六：导出与打印

| 功能 | 说明 |
|------|------|
| 尺寸预设 | 打印尺寸、朋友圈配图、桌面壁纸、社交头像 |
| 画质模式 | 高清原图 / 复古低像素 |
| 格式 | PNG / JPG |
| 打印 | 对接 Windows 系统打印 |

## 四、用户使用流程

```
打开网页 → XP桌面加载 → 主窗口自动弹出
  → ① 拍照 或 上传图片（获取4张素材）
  → ② 选择四格/拍立得模板
  → ③ 调整间距、边框
  → ④ 添加滤镜
  → ⑤ 修图（调色/贴纸/文字）
  → ⑥ 预览确认
  → ⑦ 导出保存 或 存入本地相册
  → ⑧ 以后可打开相册二次编辑
```

## 五、非功能性需求

| 类别 | 要求 |
|------|------|
| 浏览器兼容 | Chrome 90+ / Edge 90+ |
| 响应式 | 自适应 1366×768 ~ 1920×1080 主流分辨率 |
| 隐私安全 | **所有图片仅本地处理，零网络请求** |
| 性能 | 滤镜渲染 < 500ms，窗口拖拽 60fps |
| 可用性 | 无需任何安装配置，双击 HTML 即用 |
| 存储 | 使用浏览器 IndexedDB，容量取决于浏览器（通常 50MB~500MB） |
