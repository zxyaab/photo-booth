## 项目简介

一个运行在 Windows 浏览器中的**纯前端单 HTML 文件**网站，完整模拟 Windows XP 桌面环境，提供线上制作人生四格照、拍立得胶片的全套功能。所有数据仅在浏览器本地处理，不联网。

## 每次工作的标准流程

```
1. 阅读 devlog/ 最新日志
   → 了解当前进度和待办事项

2. 阅读 docs/开发步骤.md
   → 确认当前要执行哪一步

3. 阅读 docs/设计规范.md
   → 写 CSS 前对照配色表和尺寸规范，确保视觉一致

4. 阅读 docs/技术规范.md
   → 写 JS 前确认 API 用法和代码架构

5. 修改 人生四格照相馆.html
   → 所有修改只在这一个文件里进行

6. 更新 devlog/ 当日日志
   → 记录完成了什么、遇到什么问题、下一步做什么

7. 更新 docs/开发步骤.md
   → 勾选已完成的项目
```

---

## 铁律

| # | 规则 | 原因 |
|---|------|------|
| 1 | **一次只做一个步骤** | 减少 bug，方便定位问题 |
| 2 | **不要引入任何外部依赖** | 纯 HTML+CSS+JS，双击即用 |
| 3 | **所有代码写在一个 HTML 文件里** | 用户只需要一个文件 |
| 4 | **新功能不能破坏已有功能** | 每步结束要在浏览器验证 |
| 5 | **颜色/尺寸/动效对照设计规范** | 保持 XP+Y2K 视觉一致 |
| 6 | **代码注释用中文** | 方便用户理解 |
| 7 | **不联网、不调后端 API** | 隐私安全第一 |
| 8 | **每次修改后更新 devlog** | 保持开发记录完整 |
