# 🚀 一键部署 —— 让别人访问你的网站

你的网站是一个**纯静态 HTML 文件**，不需要服务器，部署到任何静态托管服务即可。

---

## 方法一：Netlify Drop（⭐ 最简单，30秒搞定）

1. 打开浏览器，访问：**https://app.netlify.com/drop**
2. 把 `人生四格照相馆.html` 文件**直接拖进去**
3. 等待 5 秒上传完，你会得到一个网址，例如：
   `https://amazing-puppy-123456.netlify.app`
4. 把网址发给朋友，他们就能打开了！

> 💡 建议注册一个免费 Netlify 账号，这样网址永久有效。不注册的话 24 小时后过期。

---

## 方法二：GitHub Pages（永久免费，推荐）

1. 打开 https://github.com ，注册/登录账号
2. 点击右上角 **+** → **New repository**
3. 仓库名填 `photo-booth`，选择 **Public**，点 **Create repository**
4. 打开终端（PowerShell），复制粘贴以下命令（把 `<你的用户名>` 换成你的 GitHub 用户名）：

```bash
cd "C:\Users\赵莘瑜\人生四格照相馆"
git remote add origin https://github.com/<你的用户名>/photo-booth.git
git branch -M main
git push -u origin main
```

5. 回到 GitHub 网页，进仓库 → **Settings** → **Pages**
6. Source 选 **main** 分支，点 **Save**
7. 等 1 分钟，你的网址就是：`https://<你的用户名>.github.io/photo-booth/`

---

## 方法三：Cloudflare Pages

1. 打开 https://pages.cloudflare.com
2. 连接 GitHub 账号
3. 选择仓库，自动部署

---

> 🎉 部署后任何人都能通过网址访问你的网站！拍照、四格合成、滤镜、贴纸全部可用。
